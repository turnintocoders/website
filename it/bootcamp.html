Bootcamp

8 settimane di allenamento super intensivo per diventare sviluppatore web full-stack con JavaScript e iniziare la tua nuova carriera in un settore in piena espansione.

Il corso comprende

8 settimane intensive. Il cuore del bootcamp sono le 8 settimane intensive di workshop e progetti, al lavoro in gruppi da 4 persone in modalità cooperativa.

Il corso copre sviluppo full-stack in JavaScript, test driven development, database relazionali e lavoro in gruppo, cenni di UX design e gestione di progetti. Puoi trovare tutti i contenuti del bootcamp e la settimana tipo nel repository Github.

Incontra le aziende. Sia durante che dopo il bootcamp avrai l'occasione di incontrare aziende alla ricerca di nuovi talenti. Ci aiuteranno revisionando il nostro codice e dando uno sguardo dal di fuori a quello che stiamo facendo.

Progetto. 8 settimane dopo il bootcamp per metterti alla prova con lo sviluppo di un progetto, insieme al tuo gruppo.

Fa per me?

Il bootcamp è il modo più veloce per diventare un programmatore. Ma prima di dedicare tempo e soldi a questa nuova avventura, devi capire se ti piace veramente e nel frattempo costruirti qualche competenza di base.

Prerequisiti

Ecco perché devi prepararti prima, da solo, partecipando ai nostri incontri settimanali Coding for everyone o entrando nella nostra chat.

Ci aspettiamo che tu completi le seguenti sezioni di FreeCodeCamp:

  • Responsive Web Design Certification
    • Basic HTML and HTML5
    • Basic CSS
    • Responsive Web Design Principles
    • CSS Flexbox
  • Javascript Algorithms And Data Structures Certification
    • Basic JavaScript
    • ES6
    • Basic Data Structures
    • Basic Algorithm Scripting
O arrivare ad un minimo di 200 punti, completando i punti precedenti più qualunque altra cosa che ti piaccia.

Dopodiché ti aspetterà la realizzazione di un piccolo sito web e un po' di pratica ulteriore in Javascript. Puoi trovare tutti i dettagli nella pagina sui prerequisiti nei materiali del bootcamp.
