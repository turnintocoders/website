Mini-bootcamp

Mini-bootcamps are a shorter version of the bootcamp, in which you can:

  • work on the bootcamp prerequisites
  • set up your developer machine
  • build your first website

where you can try the cooperative learning approach and feel what it means to code for an entire week.

Work on the bootcamp prerequisites

You can attend this mini-bootcamp to work on the prerequisites at your own pace, as we do at Coding for everyone events, but for an extended period of time.

You can prepare faster for a bootcamp or just continue the Coding for everyone experience for a longer period of time.

How long: one week, full-time, 10-18.

Next dates: 2nd October 2017, January/Februrary 2018.

Cost: zero. This round is free for all, next time will be free only for people admitted to the bootcamp.

Want to join? Apply!

Applications before: 22nd September 2017

Set up your developer machine

You've been fiddling with code for a while, maybe in FreeCodeCamp or in CodePen, but it's now time to move forward and do some coding on your computer.

To be an effective developer we recommend you to use Linux or macOS. Either way you need to learn about using the terminal, installing packages, logging into a remote server.

In this mini-bootcamp we'll set up your machine, install together the toolbox of the developer, set up a server machine over SSH and publish a minimal website.

Topics

  • Linux
  • Terminal
  • SSH
  • Set up of a web server

How long: one week, full-time, 10-18.

Next dates: 9th October 2017, January/Februrary 2018.

Cost: zero. This round is free for all, next time will be free only for people admitted to the bootcamp.

Want to join? Apply!

Applications before: 22nd September 2017
