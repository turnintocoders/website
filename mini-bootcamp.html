Mini-bootcamp

I mini-bootcamp sono una versione ridotta del bootcamp, durante i quali potrai:

  • lavorare ai prerequisiti del bootcamp
  • preparare il tuo computer per programmare
  • creare il tuo primo sito web

provando lo stile di apprendimento cooperativo e la programmazione per una settimana intera

Lavora ai prerequisiti del bootcamp

Se vuoi finire i prerequisiti alla velocità della luce vieni a questo mini-bootcamp.

Puoi venire anche solo se hai tempo di lavorare per una settimana intera sul modello di quello che facciamo a Coding for everyone.

I contenuti sono gli stessi.

Durata: una settimana, full-time, 10-18.

Prossime date: 2 ottobre 2017, gennaio/febbraio 2018.

Costo: zero.

Vuoi partecipare? Candidati!

Prepara il tuo computer per diventare programmatore

È un po' che programmi, magari su FreeCodeCamp o CodePen, ma ora vuoi fare il passo successivo e iniziare a programmare sul tuo computer.

Per diventare un programmatore web efficente ti raccomandiamo di usare Linux o macOS. Qualunque sia la tua scelta devi imparare a usare il terminale, installare pacchetti, collegarti a server remoti, ...

In questo mini-bootcamp imposteremo il tuo computer, installeremo insieme la cassetta degli attrezzi del programmatore, imposteremo un server usando SSH e pubblicheremo un semplice sito web.

Contenuti

  • Linux
  • Terminale
  • SSH
  • Impostazione di un server web

Durata: una settimana, full-time, 10-18.

Prossime date: 9 ottobre 2017, gennaio/febbraio 2018.

Costo: zero.

Vuoi partecipare? Candidati!

Crea il tuo primo sito web

Hai finito HTML&CSS su FreeCodeCamp?

Vieni a questo mini-bootcamp per fare il tuo primo sito usando Github pages!

Contenuti

  • Github pages
  • Mobile first responsive design
  • Advanced HTML&CSS

Se vuoi proseguire, aggiungere un po' di JavaScript e imparare qualcosa in più, proponiamo degli extra.

Contenuti extra

  • JSON
  • Callback

Durata: una settimana, full-time, 10-18.

Prossime date: 16 ottobre 2017, gennaio/febbraio 2018.

Costo: zero.
